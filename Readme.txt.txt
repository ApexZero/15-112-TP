Project Description:
PacManS22, it will be based off of the arcade game Pac-Man, where you control Pac-Man and try to score
the highest point by eating dots while the power ups stop the ghost from moving for a set amount of time
and during that period of time, the palyer gains double the points while eating dots.

How to run:
Run the game_basic file in the editor and the game will boot up, the ghost_astar file is just for the A* algorithm.

Libraries:
None

Shortcut Commands:
Press "r" to restart the game
Press "Space" to begin the game
Press "f" to skip to end of the game
